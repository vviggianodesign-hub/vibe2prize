About this Repository

This repository hosts the Enhanced Vibe-to-Enterprise Framework deck as a fully themeable presentation system.

  • core/: Shared engines and helpers.
  • templates/: Source MDX slides and slide-set manifests.
  • builder/: Build, lint, and content-analysis CLIs.
