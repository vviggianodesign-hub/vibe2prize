SECTION 03 // MENTAL MODELS

The 'Intern Test'

A mental model for identifying high-impact AI opportunities without the jargon.

"If you had a brilliant but inexperienced intern who could work 24/7 and never got bored—what would you have them do?"

"This is actually how people should think about AI. It gets at repetitive cognitive work without ever needing to say 'AI' or 'Machine Learning'."

Intern-Ready Tasks

01 // REPETITIVE

Tasks that follow a clear, repeatable pattern.

02 // COGNITIVE

Requires expertise but feels mechanical or routine.

03 // VERIFIABLE

Outputs that a human can easily check for accuracy.

04 // SCALABLE

Work that would be highly valuable if done at 100x speed.
